Hello!

This package consists of the following items:

* Expedited Transit Time Web Service documenation (doc folder - open or double click on index.html with any browser)
* Expedited Transit Time Web Service WSDL (TimeLookupService.wsdl - also at   www.odfl.com/wsExpTransTime/TimeLookupService/WEB-INF/wsdl/TimeLookupService.wsdl)
* Sample client code (php, Java and .Net)
* Sample XML request template

This web service is a SOAP web service. We recommend you use SOAP tools appropriate for your language to
handle making the request. The TimeLookup class has a method getTT which takes a TimeRequest object and returns a TimeResponse object.  
You will need to construct a TimeRequest and you will receive back a TimeResponse.
please see the documenation on these objects enclosed. 
	Java 1.6 JDK JAX-WS		http://docs.oracle.com/javase/6/docs/technotes/tools/share/wsimport.html
	.Net WCF			http://msdn.microsoft.com/en-us/library/ms733133.aspx
	PHP SoapClient			http://php.net/manual/en/class.soapclient.php
